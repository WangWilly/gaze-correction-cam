# Gaze-correction-cam

## Prerequisites

- https://python-poetry.org/docs/
- `AppKit`
  - https://www.freedesktop.org/wiki/Software/pkg-config/
- `dlib`
  - https://cmake.org/download/

```bash
brew install pkg-config
brew install cmake

poetry install
```

## Run the code

```bash
python regz_socket_MP_FD.py
```
